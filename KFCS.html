Kalman Filtered Compressive Sensing

Kalman filter is an important method for state estimation. However, when the number of state parameters is large, Kalman filter requires a large number of measurements. Meanwhile, compressive sensing, as an emerging technique in signal processing, allows recovering the signal though much fewer sampling points than required by the sampling theorem. Thus, by combining compressive sensing and Kalman filter, the state can be estimated through extremely low number of measurements.
However, compressive sensing can not be directly combined with the Kalman filter, and compressive sensing is based on the fact that the measured signal can be sparsely expressed by a certain system of basis functions. To address these problems, we introduce pseudo-measurement that can bridge the gap between two methods. In order to encourage the sparsity of the signal, we use the changes of the signal between time steps, rather than the signal itself, as the object of the compressive sensing. Further, we train a set of base functions, on which the target signals can have higher sparsity.

Furthermore, in order to fuse the Kalman filter and compressive sensing more deeply, we not only use the pseudo-measurement generated from the compressive sensing to facilitate the state estimation of Kalman filter, but also use the updated signal obtained by Kalman filter to improve compressive sensing to estimate the signal change at the next time step (figure 3).
